# Odin Recipes

### Description
This is my first project from [The Odin Project's Foundations course](https://www.theodinproject.com/). It is a basic recipe website built entirely with HTML to practice structuring web pages.

### Skills Demonstrated
- HTML structure (boilerplate, headings, paragraphs)
- Working with lists (ul, ol)
- Adding and managing images
- Creating internal and external links
- Basic Git/GitHub workflow (add, commit, push)
